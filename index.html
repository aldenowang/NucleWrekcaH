NucleWrekcaH

Adversarial AI Agent for Nuclear Physics Model
Neural Operator Vulnerability Discovery

NucleWrekcaH is an autonomous AI agent system designed to probe and exploit vulnerabilities in neural operator surrogate models used for nuclear physics simulation. By coupling a ReAct-loop agent with a suite of physics-based evaluation harnesses, the system systematically discovers inputs where high-fidelity machine-learning surrogates deviate from ground-truth physics — exposing reliability gaps critical to the safe deployment of AI in high-stakes scientific domains.

How It Works

⚛️
MODULE 01

Physics Simulator Evals

Six hand-crafted evaluation harnesses implement ground-truth physics solvers spanning nuclear engineering and applied mathematics:

  • Ideal Gas Law
  • Radioactive Decay
  • Separable ODEs
  • 1D Heat Conduction w/ Internal Source
  • Neutron Diffusion
  • Navier–Stokes (FNO) coming
🤖
MODULE 02

AI Agent · ReAct Loop

An autonomous agent powered by Claude operates in a Reasoning + Acting loop, iteratively forming hypotheses about model weak spots, executing targeted probes through tool calls, and updating its attack strategy based on observed error signals.

reason act observe iterate
⚡
MODULE 03

Adversarial Attack on Neural Operators

The agent targets Fourier Neural Operators and similar surrogate models, searching the input space for regions of high prediction error — inputs where the ML model's physics approximation breaks down and cannot be trusted for downstream inference or safety analysis.

The target FNO model achieves a baseline test error of 8.3% on the Navier–Stokes benchmark (Li et al., 2021).
